# Python Min Stack O(1) Implementation

This example demonstrates a Min Stack data structure implementation in Python. It extends a standard stack by providing an additional `getMin()` method, which returns the minimum element currently in the stack in O(1) time complexity. The implementation uses a secondary stack to efficiently track minimums during push and pop operations.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
